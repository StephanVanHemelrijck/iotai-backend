GET /players

To get all the existing non-guest* players, which are players that played the game without making an actual account, you need to make a GET request to the following url :
localhost:1337/players

Register user

Adviced to create a cookie to store the user info in. Read more about cookies

To create a new user and insert him into the database you need to make a POST call to the following url :
localhost:1337/player/register


BODY PARAMETERS

Field Type
name String
password String
email String

Login user

Adviced to create a cookie to store the user info in. Read more about cookies

This POST call will look for an account in the database with the email OR name provided. If that account exists, then it will check if the password given matches the password stored in the database.

To login an already existing user you need to make a POST call to the following url :
localhost:1337/login


BODY PARAMETERS

Field Type
name String Optional*
email String Optional*
password String Required
