### **.ENV File**

<hr>

Before running the application, create a new .env file in the root directory of your project and paste the following code:

```
### DATABASE CONNECTION
DB_HOST=iotai-db.c8gkzkzor7rb.eu-west-3.rds.amazonaws.com
DB_USER=admin
DB_PASSWORD=iotai123
DB_PORT=3306
DB_NAME=iotai_db

### PASSWORD HASHING
PW_SALT=9
```
